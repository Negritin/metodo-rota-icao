# Metodo Rota ICAO

Pagina de vendas para o Metodo Rota ICAO, um treinamento de ingles de aviacao para pilotos que querem estudar com rota, contexto e pratica direcionada para o ICAO.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Lucide React

## Rodar localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
